FIXimatesm

This tool is intended as a sample application for the FIX Repository. It is based on the design of FIXimatesm application that is generated from the repository but has been rewritten in XSLT. 

In the various releases of the FIX Specification, a number of fields have been replaced/deprecated. The deprecated fields are included in the messages, with a detailed explanation on the bottom frame. The replaced fields are included here for reference to previous versions. 

This tool has been developed and tested using:-

  • FireFox 1.0.2
  • Internet Explorer 6.0
  • Navigator 7.2
  • Opera 7.54u2

    • Tested as at 7 October 2006
